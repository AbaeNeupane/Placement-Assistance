This project is the job recommentdaion system and uses two algorithm **"Content based filtering via (Tf-IDF and Cosine Similarity) and Jaccard's Algorithm".** <br>
<ul>  
  <li>Has Raw Recommendation algorithm implemented.</li>  
  <li>Has Resume parsing Module.</li>  
  <li>Has Proper Authentication.</li>  
  <li>Recommended jobs to user using Content based filtering.</li> 
  <li>Recommended best candidates suitable for the specific job to recruiters using Jaccard's Coefficient.</li> 
  <li>Frontend: Html, css, bootstrap, js and jinja2.</li> 
  <li>Backend: Python Flask.</li>
  <li>Database: MongoDB or whatever you like just pull and show your creativity.</li> 
</ul>
