This project is the job recommentdaion system and uses two algorithm **"Content based filtering via (Tf-IDF and Cosine Similarity) and Jaccard's Algorithm".** <br>
Has Raw Recommendation algorithm implemented
Had Resume parsing Module
Has Proper Authentication
Recommended jobs to user using Content based filtering.
Recommended best candidates suitable for the specific job to recruiters using Jaccard's Coefficient.
Frontend: Html, css, bootstrap, js and jinja2
Backend: Python Flask
Database: MongoDB or whatever you like just pull and show your creativity.
