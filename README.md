This project is the job recommentdaion system and uses two algorithm **"Content based filtering via (Tf-IDF and Cosine Similarity) and Jaccard's Algorithm".** <br>
Has Raw Recommendation algorithm implemented <br>
Had Resume parsing Module<br>
Has Proper Authentication<br>
Recommended jobs to user using Content based filtering.<br>
Recommended best candidates suitable for the specific job to recruiters using Jaccard's Coefficient.<br>
Frontend: Html, css, bootstrap, js and jinja2<br>
Backend: Python Flask<br>
Database: MongoDB or whatever you like just pull and show your creativity.<br>
